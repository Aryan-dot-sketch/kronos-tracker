# Kronos Tracker — Desktop & Mobile Install Guide

Kronos Tracker is now a **full Progressive Web App (PWA)**.

You can install it as a real native-like app on desktop and mobile.

---

## Desktop Install (Recommended)

### Chrome / Edge / Brave (Windows, macOS, Linux)

1. Open Kronos Tracker in your browser
2. Look for the **Install** button in the top right (next to theme switcher)
3. Click **Install**
4. Confirm in the popup

**Alternative**:
- Click the **three dots** menu → **"Install Kronos Tracker"**

Once installed:
- It opens in its own window (no browser UI)
- Appears in your app launcher / Start menu / Dock
- Works offline
- Gets automatic updates

---

## Mobile Install

### Android (Chrome)
1. Open the site
2. Tap the **three dots** menu
3. Tap **"Add to home screen"** or **"Install app"**

### iOS (Safari)
1. Open in Safari
2. Tap the **Share** button (square with arrow)
3. Scroll and tap **"Add to Home Screen"**
