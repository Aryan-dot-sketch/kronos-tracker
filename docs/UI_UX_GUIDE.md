# Kronos Tracker — UI/UX Guide

## Design Direction

Kronos Tracker should feel like a premium productivity command center: cream, classic, calm, serious, and elegant.

## Color Palette

- Warm cream background: `#F7F1E8`
- Soft ivory cards: `#FFFDF8`
- Deep charcoal text: `#1E1E1E`
- Warm grey secondary text: `#6F6A60`
- Muted gold accent: `#B88A44`
- Elegant green success: `#4F7D5A`
- Amber warning: `#C8872B`
- Soft red danger: `#B85C5C`
- Dark mode base: `#15120E`

## Typography

- Heading style: premium serif feel
- Body style: clean modern sans-serif
- Numbers/stat cards: large, refined, high-contrast

## Layout

Desktop:

```text
Left Sidebar | Main Workspace | Right Insight Panel
```

Mobile:

```text
Main content + bottom navigation
```

## Component System

- Premium buttons
- Ghost buttons
- Icon buttons
- Task cards
- Stat cards
- Goal cards
- Progress rings
- Streak badges
- Timer cards
- Chart cards
- Heatmap squares
- Empty states
- Modals
- Toast messages

## Interaction Principles

- Checkboxes should feel satisfying.
- Streaks should feel motivating, not stressful.
- Missed tasks should trigger recovery, not guilt.
- Time should always feel present.
- Graphs should be useful but not overwhelming.

## Current Build Alignment

The current MVP follows the approved generated concept image: cream premium dashboard, left navigation, live IST panel, cards, checklist, charts, heatmap, and right insight panel.
